# Homework 6

## **Task 1:**  

Написать метод который принимает 2 строки - a, b. Метод должне вернуть количество символов которые у них различаются. Если строки разной длины то вернуть -1.  

**Примеры:** 
```
a="asdf", b="qwerty" => -1

a="abb", b="bba" => 0

a="abc", b="dca" => 1

a="aaa", b="fga" => 2

a="abc", b="dfg" => 3
```
